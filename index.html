Musicians' Helper - Repeater

A tool used by musicians to learn new songs.

Insert a Youtube video URL below and, by default, start time will be 0 seconds and duration will be equivalent to the full lenght of the video.
You can then change this values to configure which part of the song you want to repeat automatically. It saves a lot of time to find which note/chord is being played.

Invalid URL. Please, insert a Youtube video URL according to this format: https://www.youtube.com/watch?v={video_id}
